# v0.5 migration

## Removed product name

`frontend-craft` was the pre-v0.5 name of the canonical `design-craft` skill.
v0.5 removes the alias skill, forwarding wrappers, installer option, runtime
schema compatibility, and release checks for the former name.

Update saved prompts and local automation as follows:

```text
frontend-craft -> design-craft
frontend_craft_* -> design_craft_*
skills/frontend-craft -> skills/design-craft
```

The v0.5 installer manages only `design-craft`; it does not inspect, mutate, or
delete an existing alias. Before retiring an older installation, determine its
ownership and preserve any local changes. Historical evidence remains
immutable under `evals/**/history/` and cannot satisfy current release gates.
