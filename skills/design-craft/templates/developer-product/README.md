# Developer product seed

These files are original design-craft templates for developer tools, SaaS,
dashboards, admin systems, infrastructure products, documentation, and similar
utility surfaces that do not yet have a stronger visual authority.

- `design.md`: light design authority.
- `design.dark.md`: dark design authority with matched semantic roles.

Copy the pair into a target project, replace the product-specific placeholders,
and evolve the tokens from real runtime needs. Do not reapply the seed over an
established `DESIGN.md`, brand system, or product-specific visual language.
