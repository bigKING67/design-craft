# Vercel Geist design templates

This directory vendors the public Vercel Geist `design.md` references as local
seed templates for `design-craft`.

- `design.md`: fetched from `https://vercel.com/design.md`
- `design.dark.md`: fetched from `https://vercel.com/design.dark.md`
- Retrieved: 2026-06-24

Use these as the default initial template for new or weakly specified
developer-product, SaaS, dashboard, admin, infra, docs, and tooling surfaces.
For projects with an existing `DESIGN.md`, token system, brand guide, or strong
runtime style, treat the Geist files as a comparison baseline and adapt from the
project authority instead of overwriting it blindly.

The files are kept verbatim. Do not hand-edit them; refresh from the source URLs
when intentionally updating the bundled seed.
